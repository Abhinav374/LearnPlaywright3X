# Concept Explanation Prompt

Use this template to ask me to explain a programming/computer science concept. I'll create a detailed `.md` file in this folder with the explanation.

## Prompt

> Explain the concept of **`[CONCEPT_NAME]`** with an example from this chapter's code. Break it down simply and include comparisons, diagrams, or tables where helpful.

## Example

## How to use

1. Copy the prompt above
2. Replace `[CONCEPT_NAME]` with the concept you want explained
3. Paste it here in the chat
