# LearnPlaywright3X

A JavaScript learning repository organized by chapters, covering fundamental programming concepts.

## Chapters

- **Chapter_01** - Hello World and basic JavaScript structure
- **Chapter_02** - Variable concepts (`let`, `var`, `const`)
- **Chapter_03** - Identifier rules and naming conventions
- **Chapter_05_Operator** - JavaScript operators (arithmetic, assignment, comparison, logical)
- **IQ_Notes** - Interview questions and notes on keywords, bytecode, and binary concepts
